# アプリケーション概要
- ログイン状態でのみアプリの機能を使用できる
- 任意の月のシフトを簡単に作成できる
- 作成したシフトを簡単に編集できる
- 過去に作成したシフトはトップページにて一覧表示される
- 過去に作成したシフトは削除できる
- シフト作成をするときに便利なテンプレートが作成できる
- テンプレートは編集及び削除できる
